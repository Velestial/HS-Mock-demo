# Plan 03-04 (Gap Closure): Cancel WC Order on Failed Payment

## Gap
Criterion 3 fails: failed Stripe payments leave orphaned "pending" WooCommerce orders.

The checkout creates a WC order (pending) before charging. If Stripe rejects, the catch block resets the form but never cancels the order.

## Fix 1: Webhook handler for payment_intent.payment_failed

In `server/routes/payments.cjs`, inside the webhook route, add a handler for the failed event alongside the succeeded handler:

```js
if (event.type === 'payment_intent.succeeded') {
  // ... existing succeeded logic ...
}

if (event.type === 'payment_intent.payment_failed') {
  const paymentIntent = event.data.object;
  const wcOrderId = paymentIntent.metadata?.wc_order_id;
  if (wcOrderId) {
    try {
      const api = require('../woocommerce.cjs');
      await api.put(`orders/${wcOrderId}`, { status: 'cancelled' });
      console.log(`[webhook] Order ${wcOrderId} cancelled after payment failure`);
    } catch (err) {
      console.error(`[webhook] Failed to cancel order ${wcOrderId}:`, err.message);
    }
  }
}
```

## Fix 2: Client-side fallback in CheckoutPage catch block

In `components/CheckoutPage.tsx`, the catch block at the end of `handleSubmit` (around line 216-222) currently does:
```tsx
} catch (err: any) {
  console.error("Checkout Error:", err);
  const backendError = ...;
  setError(...);
  setStep('form');
}
```

We need to track the WC order ID outside the try block so the catch can cancel it. Refactor to:

1. Add a `let wcOrderId: number | null = null;` before the try block
2. After `const order = await createOrder(orderData);`, set `wcOrderId = order.id;`
3. In the catch block, if `wcOrderId` is set and the error is Stripe-related (not the order creation itself), cancel the order:

```tsx
} catch (err: any) {
  console.error("Checkout Error:", err);
  // Cancel the WC order if it was created but payment failed
  if (wcOrderId) {
    try {
      await updateOrderStatus(wcOrderId, 'cancelled');
    } catch (cancelErr) {
      console.error("Failed to cancel order:", cancelErr);
    }
  }
  const backendError = err.response?.data?.details || err.response?.data?.error || err.message;
  setError(typeof backendError === 'string' ? backendError : JSON.stringify(backendError) || "An unexpected error occurred.");
  setStep('form');
}
```
