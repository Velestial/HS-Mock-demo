## Task 2: Remove `any` from services/api.ts

Read `services/api.ts`. Find all parameters or return types annotated as `any` or using implicit `any`.

Key areas to type:
- `createOrder(orderData: any)` → define an `OrderPayload` interface
- `updateCustomer(customerId: number, data: any)` → define a `CustomerUpdatePayload` interface

Add these interfaces before the functions:
```ts
interface OrderLineItem {
  product_id: number;
  name: string;
  price: string;
  quantity: number;
}

interface OrderShippingLine {
  method_id: string;
  method_title: string;
  total: string;
}

interface OrderAddress {
  first_name: string;
  last_name: string;
  address_1: string;
  address_2?: string;
  city: string;
  state: string;
  postcode: string;
  country: string;
  email?: string;
  phone?: string;
}

interface OrderPayload {
  payment_method: string;
  payment_method_title: string;
  set_paid: boolean;
  billing: OrderAddress;
  shipping: OrderAddress;
  line_items: OrderLineItem[];
  shipping_lines: OrderShippingLine[];
}

interface CustomerUpdatePayload {
  first_name?: string;
  last_name?: string;
  email?: string;
  billing?: Partial<OrderAddress>;
  shipping?: Partial<OrderAddress>;
}
```

Update function signatures:
- `createOrder(orderData: OrderPayload)`
- `updateCustomer(customerId: number, data: CustomerUpdatePayload)`
